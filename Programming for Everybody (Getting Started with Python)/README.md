# Programming for Everybody (Getting Started with Python)

This repository contains my completed programming assignments for the 
University of Michigan course **"Programming for Everybody (Getting Started with Python)"** on Coursera.

This course focuses on building foundational programming skills using Python 3 and emphasizes computational thinking, structured logic, and problem-solving.

---

## Course Scope

The course covers fundamental programming concepts including:

- Variables and Expressions
- Conditional Logic
- Functions
- Loops and Iteration
- Exception Handling
- Basic Input/Output
- Program Structure and Modularity

All assignments were implemented independently and executed locally using Python 3.

---

## Repository Structure

Each graded programming assignment is organized as follows:

- `.py` file — Source code implementation
- `.png` file — Screenshot of program execution output

Example:

- `2.3 Pay Calculator.py` 
- `2.3 Pay Calculator.png`
